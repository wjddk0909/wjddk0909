## git branch 생성하고 사용하는 방법

- git branch : 깃 브랜치 확인
- git branch -v : 마지막 커밋메세지 확인
- git branch 브랜치명 : 브랜치 생성
- git checkout 브랜치명 : 선택한 브랜치로 이동
- git add . : 변경한 파일을 저장
- git commit -m "커밋메세지" : 커밋메세지 입력
- git push origin 브랜치명 : 원격저장소 브랜치에 push 하기
