## 1.1 npm 초기화 명령어 - init

- npm init 으로 package.json을 생성
- 또는 npm init -y 로 자동 생성

## 1.2 npm 설치 명령어 - install

- npm install jquery 로 설치하면 node_modules폴더 생성되고 그 안에 라이브러리가 설치됨
- packge.json에도 jquery의 버전도 명시 되어 있음
